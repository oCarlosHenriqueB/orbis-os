Política de Privacidade

Orbis OS — Gerenciador de Ordens de Serviço · Atualizado em junho de 2025

1. Sobre o Aplicativo

O Orbis OS é um sistema de gerenciamento de ordens de serviço desenvolvido exclusivamente para uso interno na Engenharia Clínica do Hospital e Maternidade Municipal Santa Ana, localizado em Santana de Parnaíba — SP. O aplicativo é destinado aos técnicos e profissionais da instituição.

2. Dados Coletados

O aplicativo coleta e armazena apenas os dados necessários para seu funcionamento:

  • Autenticação: endereço de e-mail e senha (gerenciados pelo Firebase Authentication).
  • Ordens de Serviço: número, equipamento, setor, tipo de manutenção, descrição do problema e solução, técnico responsável e datas.
  • Equipamentos: nome, marca, modelo, NTag/patrimônio, setor e criticidade.
  • Preferências locais: nome do técnico, armazenado apenas no dispositivo via SharedPreferences.

3. Uso dos Dados

Os dados são utilizados exclusivamente para:

  • Gerenciamento interno de ordens de serviço hospitalares.
  • Controle e rastreabilidade de manutenções de equipamentos biomédicos.
  • Geração de relatórios operacionais para uso interno.

Nenhum dado é compartilhado com terceiros, vendido ou utilizado para fins publicitários.

4. Armazenamento e Segurança

Os dados são armazenados no Firebase Firestore (Google Cloud), com acesso restrito por autenticação. Apenas usuários autorizados pela instituição possuem credenciais de acesso ao sistema. As senhas são gerenciadas pelo Firebase Authentication e nunca são armazenadas em texto simples.

5. Acesso e Exclusão de Dados

Por se tratar de um sistema de uso interno hospitalar, o acesso e a exclusão de dados são gerenciados pelo administrador do sistema da instituição. Em caso de solicitação, entre em contato com a Engenharia Clínica do hospital.

6. Serviços de Terceiros

O aplicativo utiliza os seguintes serviços do Google:

  • Firebase Authentication — autenticação de usuários.
  • Cloud Firestore — banco de dados em nuvem.

Consulte a Política de Privacidade do Google para mais informações.
